# Character Encodings

- ASCII
  - 符号化文字集合(ex.あ -> U+3042)
- Unicode
  - 符号化文字集合(ex.あ -> U+3042)
- Shift JIS
  - 文字符号化方式
- EUC-JP
  - 文字符号化方式
- UTF-8
  - 文字符号化方式(ex.あ -> 11100011 10000001 10000010 -> E3 81 83) ※最右は16進数表記
- UTF-16
  - 文字符号化方式(ex.あ -> 00110000 01000010 -> 3042) ※最右は16進数表記

- URLエンコーディング（パーセントエンコーディング）
